常见问题

data() 函数是否被废弃？

data() 函数并没有被废弃，Vue 3仍然支持Options API，其中包括 data() 函数。但在使用Composition API时，不再使用 data() 函数。

ref() 和 reactive() 的区别是什么？

  • ref() 用于包装非对象值（如数字、字符串等），使其具有响应式特性。访问 ref 的值时需要使用 .value 属性。
  • reactive() 用于包装对象，使其具有响应式特性。访问 reactive 对象的属性时不需要使用 .value 属性。
