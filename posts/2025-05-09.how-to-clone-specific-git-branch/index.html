最佳实践

  • 如果只需要克隆一个分支，且仓库较大，建议使用 --single-branch 选项，避免下载不必要的分支数据。
  • 在克隆仓库后，可以使用 git branch -a 查看所有分支，确保切换到正确的分支。

常见问题

  • Bash 补全不支持 --branch 选项：这是一个已知问题，目前没有直接的解决办法，但不影响命令的正常使用。
