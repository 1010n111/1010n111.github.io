2 Docker容器使用

命令 作用 参数
docker ps 查看正在运行的容器
docker stop 停止容器
docker start 启动容器
docker rm 删除容器
docker run 运行容器 -d：后台 -P：容器内部的网络端口映射到主机
docker top 查看容器内部运行的进程
docker inspect 查看容器的配置和状态信息

3 Docker镜像使用

命令 作用 参数
docker images 列出本地主机镜像
docker search 搜索远程仓库镜像
docker pull 下载镜像
