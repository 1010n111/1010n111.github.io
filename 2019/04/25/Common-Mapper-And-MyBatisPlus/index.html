通用Mapper

  • springboot

2. DAO

  • DAO接口继承Mapper接口,并表明泛型

3. 数据库映射

  • 默认情况根据 驼峰命名法 自动进行映射

    3.1 @Table 注解

    实体类类上

属性名 属性值
name 表名
catalog 目录(MySql不支持)
schema 数据库名(MySql支持)

catalog 与 schema 同时设置时, catalog 的优先级高于 schema,也就是只有 catalog 生效

3.2 @Column 注解

实体类属性

属性名 属性值 说明
name 列名
insertable true/false 设为false不会出现在insert sql语句中
updateable true/false 设为false不会出现在update sql语句中

3.3 @ColumnType注解

实体类属性

属性名 属性值 说明
column 列名 和@column注解的name属性作用相同，但是@Column注解优先级更高
jdbcType JdbcType.XXX 设置特殊数据库类型时指定指定数据库中的 jdbcType
typeHandler 设置特殊类型处理器

3.4 @Transient注解

实体类属性
告诉通用Mapper，该属性不是数据库中的字段

3.5 @Id注解

实体类属性
标明该属性为 主键 或 联合主键

3.6 @KeySql注解

实体类属性
主键策略注解

  • 自增(以MySql为例)
  • UUID(以MySql为例)
属性名 属性值 说明
useGeneratedKeys true/false 是否使用JDBC方式获取主键，优先级最高
dialect IdentityDialect.XXX 根据配置的数据库类型取回主键，优先级第二
sql SQL语句 取主键的SQL语句
order ORDER.XXX 取主键的语句执行的顺序顺序
