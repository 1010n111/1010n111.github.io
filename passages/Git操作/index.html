Git操作

Git安装

打开Git-2.36.1-64-bit.exe,

一直点击next

Git GUI软件安装

  1. 打开 TortoiseGit-2.13.0.1-64bit.msi 安装GUI软件
  2. 打开 TortoiseGit-LanguagePack-2.13.0.0-64bit-zh_CN.msi 安装GUI的汉化文件

配置账号

打开设置

设置名称与邮箱

使用GUI

  • 以现有的文档库为例

http://git.t.easydcmm.com/easydata/file

  • 将IP替换为域名

http://192.168.1.10/easydata/file.git –> http://git.t.easydcmm.com/easydata/file.git

  • 克隆仓库

选择 Git 克隆

调整URL和目录,点击确定

  • 更新文件

在文件目录内点击拉取,更新文件

  • 推送文件

    先提交,再推送

    选择Git提交

    填写更改说明,选择变更文件,点击提交

    推送到服务器

提交成功后,点击推送
